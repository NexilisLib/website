FAQ

What is the goal of Nexilis?

Easy multiplayer game development with TCP and UDP connection.

Why isn't it released?

Nexilis is simply not ready. Unstable release is in the future plans.

Will there be bindings to another languages?

C# is planned in release. Others are a possibility.

What is the future of Nexilis?

  • 3D functionality for client api
  • Expanding C# api
  • Client development with Unity
  • Still a lof of core library work
  • Improving currently very limited testing
  • Open sourcing
